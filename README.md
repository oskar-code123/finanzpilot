# FinanzPilot - Immobilienfinanzierung leicht gemacht

Interaktiver Immobilienfinanzierungsrechner mit Schritt-fuer-Schritt-Wizard, Tilgungsplan-Visualisierung und Szenario-Vergleich fuer alle deutschen Bundeslaender.

## Features

- Kaufnebenkosten-Berechnung (Grunderwerbsteuer, Notar, Makler)
- - Annuitaetenberechnung mit Tilgungsverlauf
  - - Zinssatz-Ermittlung basierend auf Eigenkapitalquote, Zinsbindung und Bonitaet
    - - Szenario-Vergleich (bis zu 4 Finanzierungen nebeneinander)
      - - Eigenkapital-Wirkungsanalyse
        - - Anschlussfinanzierungs-Simulation
         
          - ## Quickstart
         
          - ```bash
            npm install
            npm run dev
            ```

            ## Hinweis

            Unverbindliche Beispielrechnung - keine Finanzberatung.
